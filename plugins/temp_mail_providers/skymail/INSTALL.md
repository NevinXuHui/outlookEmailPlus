# SkyMail 插件安装说明

### 单文件版本（已安装）

插件已安装为单个 Python 文件：
\`\`\`
/root/outlookEmailPlus/plugins/temp_mail_providers/skymail.py
\`\`\`

如需查看完整文档，文件位于：

/root/outlookEmailPlus/plugins/temp_mail_providers/skymail/
